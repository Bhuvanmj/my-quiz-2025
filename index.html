Data Communication Quiz

1. What is the main goal of data communication in computer networks?
a) Printing documents
b) Sharing multimedia only
c) Reliable exchange of data between devices
d) Playing games
Correct Answer: c) Reliable exchange of data between devices
2. Which mode allows data to flow in both directions, but only one at a time?
a) Simplex
b) Half-Duplex
c) Full-Duplex
d) Analog
Correct Answer: b) Half-Duplex
3. Which transmission medium uses light signals for data transmission?
a) Twisted Pair
b) Coaxial Cable
c) Optical Fiber
d) Infrared
Correct Answer: c) Optical Fiber
4. Which protocol is mainly used for sending and receiving email?
a) HTTP
b) FTP
c) TCP/IP
d) SMTP
Correct Answer: d) SMTP
5. What type of data transmission sends one bit at a time and is good for long distances?
a) Parallel
b) Serial
c) Analog
d) Digital
Correct Answer: b) Serial
6. Which network device broadcasts data to all devices in a network?
a) Switch
b) Hub
c) Modem
d) Router
Correct Answer: b) Hub
7. Which device directs data between different networks?
a) Switch
b) Router
c) Hub
d) Modem
Correct Answer: b) Router
8. What protocol breaks data into packets and routes them for reliable delivery?
a) FTP
b) TCP/IP
c) SMTP
d) HTTP
Correct Answer: b) TCP/IP
9. Which wireless medium requires line-of-sight communication?
a) Radio
b) Microwave
c) Infrared
d) Wi-Fi
Correct Answer: b) Microwave
