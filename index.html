Data Communication Quiz

1. What is the main purpose of data communication?
a) Playing games
b) Exchanging data between devices
c) Making phone calls only
d) Printing documents
Correct Answer: b) Exchanging data between devices
2. Which mode allows communication in both directions at the same time?
a) Simplex
b) Half-Duplex
c) Full-Duplex
d) None of these
Correct Answer: c) Full-Duplex
3. Which device connects different networks together?
a) Router
b) Hub
c) Modem
d) Keyboard
Correct Answer: a) Router
4. Which protocol is used for loading websites?
a) SMTP
b) HTTP
c) FTP
d) TCP
Correct Answer: b) HTTP
5. Which cable uses light to send data?
a) Twisted Pair
b) Coaxial Cable
c) Optical Fiber
d) USB Cable
Correct Answer: c) Optical Fiber
6. What does TCP/IP do?
a) Sends emails
b) Breaks data into packets for sending
c) Connects to Wi-Fi
d) Makes phone calls
Correct Answer: b) Breaks data into packets for sending
7. Which is NOT a network device?
a) Switch
b) Router
c) Toaster
d) Hub
Correct Answer: c) Toaster
8. Which mode is like a walkie-talkie?
a) Simplex
b) Half-Duplex
c) Full-Duplex
d) None of these
Correct Answer: b) Half-Duplex
9. What is the fastest wired transmission medium?
a) Twisted Pair
b) Coaxial Cable
c) Optical Fiber
d) Power Line
Correct Answer: c) Optical Fiber
10. Which protocol is used for sending emails?
a) HTTP
b) FTP
c) SMTP
d) TCP
Correct Answer: c) SMTP
